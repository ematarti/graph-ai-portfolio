About the course

8‑week live cohort • No‑code AI tools • Responsible task automation

GRAPH Courses

What I’m learning

The curriculum is designed to improve productivity in research and operations through responsible use of GenAI. Across weekly workshops, help sessions, feedback, and a capstone, the course emphasizes transparent workflows and high‑quality outputs.

Bootcamp features

  • Weekly live workshops + live help sessions
  • Private community forums
  • Personalized feedback on assessments
  • Capstone project coaching
  • Shareable certificate + lifetime tutorial access
Module 1 — Websites, Documents & Presentations LLMs • Markdown • Canvas

Learn how LLMs work, generate text responsibly, format documents efficiently, and design clear presentations/posters.

  • LLM training basics and major providers
  • Generate Markdown, LaTeX, and HTML outputs
  • Build simple research websites with ChatGPT Canvas and no‑code tools
  • Use Markdown for structured research documents
Module 2 — AI‑Automated Data Analysis Dashboards • Reports

Use AI tools responsibly for analysis, file manipulation, and generating accurate automated reports.

  • Quick data analysis with code interpreters
  • Automated dashboards and recurring spreadsheet workflows
  • Dataset processing, transformations, and batch conversions
  • Transparent visualizations and report generation
Module 3 — GenAI for Research & Text Analysis Lit review • Synthesis

Accelerate literature search and knowledge synthesis while keeping methods transparent and responsible.

  • Literature reviews with tools like Elicit / LiteRev
  • Text classification and document organization
  • Summarization workflows and knowledge mapping
  • Information synthesis systems for research topics
Module 4 — Custom GPTs & Chatbots RAG • Integrations

Build chat assistants for research support and knowledge management, including retrieval‑augmented workflows.

  • Chat architecture, context windows, and RAG systems
  • Custom GPTs for specific use cases
  • Interactive chatbots and knowledge base integration
Module 5 — AI for Task Automation Agents • Workflows

Design responsible AI agents for repetitive tasks and orchestrate app‑to‑app automation flows.

  • Agent‑based automation fundamentals
  • Email and browser‑based automation patterns
  • Complex automation flows for research monitoring
Module 6 — GenAI for Communication & Outreach Figures • Storytelling

Create authentic dissemination assets: summaries for different audiences, visuals, and explanatory content.

  • Transparent research summaries
  • Figures, diagrams, and prompt‑based visualization
  • Video abstracts / explanations and content scheduling
Module 7 — Capstone Project & Showcase Demo • Feedback

Plan, build, and present a final solution with peer and instructor feedback.

  • Progress checks + peer feedback sessions
  • Final presentations with Q&A
  • Responsible AI practice showcase
Grading rubric
  • Online quizzes: 15%
  • Workshop assignments: 35%
  • Attendance & participation: 10%
  • Final project: 40%
Course expectations
  • Passing grade: 80%
  • Attend at least 5 workshops for full attendance credit
  • Focus on ethics, accuracy, and transparency in automation
